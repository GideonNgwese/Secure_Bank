# App logo

Drop your logo here as **`logo.png`** (this exact filename):

```
assets/logo/logo.png
```

It will automatically appear at the top of the Welcome, Login, Register and
Verify-Email screens via the `AppLogo` widget. A transparent-background PNG
(roughly 512×512 or a wide wordmark) works best.

Until you add `logo.png`, a shield + "SecureBank" placeholder is shown instead.

Optional: you can also add `logo_dark.png` / different sizes later and update
`lib/core/widgets/app_logo.dart` if you want variants.
